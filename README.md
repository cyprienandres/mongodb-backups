## Config file

- `backups`: array
    - `name`: backup name
    - `schedule`: cronJob schedule. Example: `0 * * * *`
    - `mongodb`:
        - `host`: MongoDB host
        - `port`: MongoDB port
- `bucket`: dictionary
    - `s3`:
        - `name`: bucket name
        - `region`: bucket region
